1.背景介绍

费舍尔耶茨洗牌

其实它的思想非常的简单，反复从数组中抽出一个随机元素并将其插入新数组，只要以相同的概率从数组中挑选每个剩余的元素，当完成时，将获得完全无偏的随机数组

2.知识剖析

双数组

按照洗牌的思路，我们在原数组中随机抽取一个元素插入新的数组，然后移除原数组中的元素

function shuffle(a) {
var b = [ ];

while (a.length) {
var index = ~~(Math.random() * a.length);
b.push(a[index]);
a.splice(index, 1);
}br
return b;
}
这个解法的正确性应该是没有问题的。我们假设数组的元素为 0 - 9，对其乱序 N 次，那么每个位置上的结果加起来的平均值理论上应该接近 (0 + 9) / 2 = 4.5，且 N 越大，越接近 4.5。

3.常见问题

性能问题

上面的方法中用了 splice，如果把 splice 的复杂度看成是 O(n)，那么整个程序的复杂度是 O(n^2)。

4.解决方案

数组内交换
洗牌元素的数量加上剩余元素的数量总是等于数组全部元素。这意味着我们可以在没有任何额外空间的情况下进行整个随机洗牌！我们使用数组的后面来存储随机抽取出的元素，并使用数组的前面来存储剩余的元素。只要我们在抽取式时保持相同的概率，我们可以不关心剩余元素的顺序！

5.代码实战

从数组中抽出一个随机元素并将其保存在一个变量中，然后把数组最后一个元素赋值给随机抽取到的元素，之后把之前保存了随机元素的变量赋值给最后一个元素，完成两个元素的交换后，把最后一个元素排除在外继续进行交换，直到进行到数组的第一个元素。
